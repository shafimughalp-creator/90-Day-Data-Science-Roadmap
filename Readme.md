## Currently Learning

- **Descriptive Statistics** — mean, median, std, variance, skewness, percentiles, IQR with NumPy
- **Probability & Distributions** — conditional probability, normal/binomial distributions, CLT
- **Outlier Detection** — IQR method in practical data cleaning pipelines
- **Correlation Analysis** — Pearson, Spearman; multicollinearity detection for feature selection
- **Hypothesis Testing** — p-values, t-tests, significance levels with scipy.stats
- **NumPy** — vectorized operations, broadcasting, matrix math for DS (Phase 2)
- **Pandas** — DataFrames, cleaning, GroupBy, merge, apply, pivot tables (Phase 2)
- **SQL** — SELECT through Window Functions, CTEs, Python integration via sqlite3 (Phase 3)
- **Machine Learning** — Linear/Logistic Regression, Decision Trees, Random Forest, XGBoost (Phase 4)
- **Deployment** — Streamlit apps hosted on Streamlit Cloud (Phase 5)
