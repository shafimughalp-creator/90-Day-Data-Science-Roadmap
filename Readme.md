## 📖 About Me

I'm a 2nd-semester **BS Artificial Intelligence** student at **Emerson University Multan, Pakistan**, working through a self-built **90-Day Data Science Roadmap** — no shortcuts, math and fundamentals before models.

I learn by building in public: every day ends with a GitHub commit, a notebook, and a LinkedIn post. I use football data as my applied-learning framework — every statistical or ML concept gets tested against a synthetic **FC Lahore Lions** dataset before it's considered "learned."

**Currently:** ✅ Phase 1 (Statistics & Math) complete · 🟢 Phase 2 (NumPy · Pandas · Visualization) in progress

**Open to:** Data Analyst / Data Science internships, freelance data-analysis work, and collaboration on beginner-friendly DS projects.
