The backbone of every DS algorithm. Most beginners skip this — don't. You'll thank yourself in every interview and every project.

Phase 02

Data Tools: NumPy · Pandas · Visualization

The daily toolkit of every data scientist. Learn these deeply — not just syntax but the thinking patterns behind them.

Days 16–3520 days · ~40 hrs
🛠️NumPy gets 4 days, Pandas gets 9 days, Visualization gets 4 days — because real DS work is 70% data wrangling. Do not rush these.
Phase 03

SQL for Data Science

SQL is asked in 9 out of 10 DS interviews. Learn it properly — from SELECT to window functions to running queries inside Python.

Days 36–4712 days · ~24 hrs
🗄️We go beyond basic SELECT. Window functions, CTEs, subqueries, and connecting SQL to a pandas pipeline are all covered.
Phase 04

Every algorithm gets two things: a theory day (understand it deeply) and a code day (apply it correctly). No blind sklearn button-pressing.

Days 48–7528 days · ~56 hrs
